C

기초프로젝트I, 알고리즘, 데이터 구조 등 C를 사용하는 수업을 수강하였습니다.
프로젝트
팀 프로젝트로 터미널 상에서 작동하는 POS를 만들었습니다.

사용 가능한 개발도구

Visual Studio Code

주로 자바 프로그래밍을 할 때 사용합니다. java를 터미널로 사용 하는 방법을 먼저 배워 eclipse보다 편하게 느낍니다.

Github

2019년도 오픈소스에 대한 강의를 들으면서 사용하게 되었습니다. 그당시 너무 활발하게 활동 하여 아이디가 한번 정지된 뒤로 새로운 아이디는 조용하게 사용하고 있습니다.

Android Studio

학교 팀 프로젝트를 위해 앱 개발을 하면서 자연스럽게 학습히게 되었습니다.

Vim

학부생 신분으로 연구실 생활을 하면서 Ubuntu를 다룰 일이 있어 간단한 편집을 할 수 있는 정도로 다룰 수 있습니다.

프로젝트

Garam

팀 프로젝트
사용 언어 : C
내용 : 식당 POS
역활 : 테이블을 선택하여 주문을 받고 주문 받은 매뉴의 수와 결제 금액을 출력해주기
개발 환경 : Visual Studio 2016
특이사항 : sutjjang라는 아이디를 하용하다 정지를 당해 현제 아이디와 다릅니다.

https://github.com/sudo-G41/garam.git

Github-overview-data

개인 프로젝트
사용 언어 : java
내용 : Github 활동량 측정 및 출력
역활 : 전부
개발 환경 : Visual Studio Code
특이사항 : github에서 주는 API를 사용하여 해당 아이디의 6가지(활동량, Repositories 개수, 자신의 Repositories가 스타&포크 된 횟수, 팔로잉 수, 팔로워 수)를 측정합니다.

https://github.com/sudo-G41/Github-overview-data.git

Todo-List

팀 프로젝트
사용 언어 : java
내용 : 할 일 리스트 앱
역활 : Firebase를 사용한 공유기능, 공유된 일정을 보기 쉽게 하기 위한 캘린더 제작, API를 사용한 한국어 & 일본어 지원
개발 환경 : Android Studio
특이사항 : Firebase를 이용하여 일정을 공유 할 수 있고 Naver API를 이용하여 일본어와 한국어 둘다 사용을 할 수 있도록 만들었습니다.

https://github.com/sudo-G41/Todo-List.git
